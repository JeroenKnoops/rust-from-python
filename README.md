# Rust-from-python

Code from [https://doc.rust-lang.org/1.3.0/book/rust-inside-other-languages.html](https://doc.rust-lang.org/1.3.0/book/rust-inside-other-languages.html).

## Compile Rust code

```
cargo build --release
```

## Run python

Python version now only supports osx.

```
python embed.py
```
